# Registro semanal de ventas

### Se tienen las ventas diarias de 5 días para 4 productos diferentes almacenadas en una matriz double[,]. Diseñar un programa que:

1. Lea las ventas desde el teclado utilizando **una matriz** de 5 filas (días) y 4 columnas (productos)

2. Calcule el total de ventas por cada día (suma de cada fila)

3. Calcule el total de ventas por cada producto (suma de cada columna)

4. Determine el día con mayor cantidad de ventas y muestre su índice y valor total

5. Determine el producto más vendido y muestre su índice y total acumulado

6. Calcule el promedio general de todas las ventas registradas

**Bonus:** Cuente cuántas ventas individuales superan el promedio general
